=== Calculate BOINC throughput data ===

Usage: get_throughput_data.py <start date YYYY/MM/DD> <end date YYYY/MM/DD>

What is it?
------------
This is a simple python script to calcuate throughput data for a BOINC grid computing server. The script outputs two files - one formatted for graphing with gnuplot and one with relevant information in human readable format. Also included is a gnuplot script for creating stacked histograms with the data.

Running the script
------------------
Before running the script, make sure to enter the correct name, username, and password for *your* BOINC mySQL database.
